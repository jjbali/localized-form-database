Backup & Restore

What does export/import do?
Database export/import creates a literal, full-fidelity copy of your database(s) — every field of every record (including formatted messages and custom fields) is preserved exactly, not simplified into plain text. Use "Export Database" to save just the database selected above as a single .json file you can move to another device; use "Export All Databases" to back up everything at once. "Import Database(s)" reads such a file back in — it always creates brand-new database(s) alongside whatever is already on this device, so nothing existing is ever overwritten.

Where is my data stored?
The data is stored locally, meaning there is no server involved or someone that will mess with your data, you can download all of your saved data below. If you're not the only one using this device, please delete your data after downloading all the records.
