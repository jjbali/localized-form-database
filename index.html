Backup, Restore & About

Backup & Restore

Database export/import creates a literal, full-fidelity copy of your database(s) — every field of every record (including formatted messages, custom fields, and the dictionary) is preserved exactly, not simplified into plain text. Use "Export Database" to save just the database selected above as a single .json file you can move to another device; use "Export All Databases" to back up everything at once. "Import Database(s)" reads such a file back in — it always creates brand-new database(s) alongside whatever is already on this device, so nothing existing is ever overwritten.

Spreadsheet

Saved spreadsheets live in this database alongside your records and dictionary — open, edit, or create as many as you like, and switch between them anytime. Type a formula starting with = — arithmetic (=A1+B2*2, =2^10), comparisons (=A1>10), text concatenation (=A1&" "&B1), and functions like SUM, AVERAGE, IF, AND/OR, VLOOKUP, INDEX/MATCH, SUMIF/COUNTIF, and common text functions (CONCATENATE, LEFT/RIGHT/MID, UPPER/LOWER, LEN). Excel import/export preserves formula text, but any Excel function not in this list will show #NAME? until edited.
A1
Click a cell, or click-and-drag (or shift+click) to select a range, then use the toolbar above to format it. Drag the small square at the bottom-right corner of a selection to autofill — numeric sequences continue the pattern, formulas adjust their relative references, and anything else repeats. Drag the thin edge on a column or row header to resize it.
Tip: hover any row or column header and use its × to delete that specific row/column.

Math Question Generator

Generates practice math questions entirely in-app — nothing is sent anywhere. Fractions, exponents, and square roots are formatted with plain HTML/CSS locally, so it works fully offline. The chosen difficulty — including a Custom min/max range — applies to every question type, with no hidden caps. Statistics topics (Mean, Median, Mode, Range, Quartile, Decile, Percentile) generate a random data set and ask you to compute the requested value; quartiles use the median-of-halves method and percentiles/deciles use the nearest-rank method, both common classroom conventions.

Terminal

A little offline hacking-style minigame — explore a fake filesystem with ls, cd, and cat, find passwords hidden in files to unlock locked directories, and track down the flags. Clear every flag in a round and the mainframe regenerates a brand new, harder layout just for you — it never ends. Your round, total flags collected, score, and Battle Pass are saved to your account. Everything is simulated locally; no real files or network are touched. Type help to see every command.
Score: 0
Shop — spend your score
Upgrades are permanent and carry over every round. Prices go up each time you buy the same upgrade again.
Battle Pass
Every point you earn anywhere in the terminal — reading files, unlocking directories, capturing flags, clearing rounds — also earns that many Battle Pass XP. Leveling up pays out a one-time bonus of spendable score and a rank like "Recruit 1". Each tier (e.g. Recruit, Initiate, Operative...) spans several levels before you're promoted to the next one. It never caps out; once every tier's been used it starts over at a higher numeral (e.g. "Legend II 1").
Community Levels — build & play custom levels
Anyone can design their own fake filesystem — directories, files, locked vaults, flags — and publish it here for everyone else to explore. Playing (or building) a community level is purely for fun: it never touches your score, rank, or Battle Pass. Type newlevel <name> below or in the terminal to start building (then mkdir/touch/write/flag/lockdir/ motd to shape it and savelevel to publish), or just pick one from the list and jump in with playlevel <id>.

Terminal Game — Admin

This is a never-ending game: once a player clears every flag in their current round, the mainframe procedurally generates a new, harder layout just for them, forever. There's a shared round 1 — the starting world every new player sees — and it's no longer edited as raw JSON. Instead, build it live in the terminal above using admin-only commands: editfs to enter edit mode, then mkdir, touch, write, rm, flag, lockdir, unlockdir and motd to shape it, savefs to publish it as round 1 for every new player, and stopedit to return to your own game. Type help in the terminal while logged in as admin to see the full list.

Scoring

Point values players earn for each action. These are plain numbers you set here — no filesystem JSON editing needed. Changes apply to players' next action.

Shop Pricing

Base cost and effect size for each shop upgrade. Actual cost rises each level a player buys (base × 1.6 per level already owned).

Battle Pass

Every point a player earns anywhere in the game also earns that much Battle Pass XP. Set how much XP level 1 costs and, optionally, a % by which the XP cost grows every level after that (0% keeps every level the same cost; e.g. 10% means level 2 costs 10% more XP than level 1, level 3 costs 10% more than level 2, and so on) — plus the one-time score bonus paid out per level-up (base + this much extra per level). Tiers are awarded in order, one per line below — once the list runs out it repeats with a higher numeral tier (e.g. "Legend II 1").
Each line is a tier name. Add :N to make a tier span N levels, shown as "Recruit 1", "Recruit 2" ... "Recruit N" before moving to the next tier — e.g. Recruit:3 gives 3 Recruit levels. Leave off :N for a single-level tier. Edit the number to change how many levels any tier lasts.

Ranks

Players are promoted automatically as their cumulative score crosses each threshold below, with a promotion banner printed in their terminal. The rank with the lowest threshold is everyone's starting rank, so keep at least one rank at 0 points.
